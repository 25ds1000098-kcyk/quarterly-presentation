ðŸ§® Financial Model: Capital Asset Pricing Model (CAPM)

We use CAPM to estimate the expected return of an asset, $E(R_i)$.

$$E(R_i) = R_f + \beta_i (E(R_m) - R_f)$$

Where:

  • $R_f$: Risk-free rate (e.g., Treasury bond yield).
  • $\beta_i$: Beta, a measure of systematic risk.
  • $E(R_m)$: Expected market return.
  • $(E(R_m) - R_f)$: Market Risk Premium.

Simplified Python function for calculating credit exposure:


def calculate_exposure(loan_amount: float, default_probability: float) -> float:
    """Calculates Expected Loss (EL)."""
    # Assuming Loss Given Default (LGD) is 40% (0.4)
    loss_given_default = 0.4
    expected_loss = loan_amount * default_probability * loss_given_default
    return expected_loss

# Example usage
exposure = calculate_exposure(loan_amount=100000, default_probability=0.05)
